# OXY — сайт проекта

## Что сделано
Собрана статическая посадочная страница с обязательными материалами поддержки:
- логотипы Фонда содействия инновациям и мероприятия «Платформа университетского технологического предпринимательства» на главной;
- точный текст поддержки на странице;
- пример брендирования фото/видео (с логотипами).

## Как запустить локально
Откройте `index.html` в браузере. Для корректного отображения ресурсов используйте веб-сервер (например, встроенный в редактор) или раздайте папку через любой локальный сервер.

## Что нужно заменить
1. Замените заглушки:
   - `assets/logo-fii.svg`
   - `assets/logo-platform.svg`
   - `assets/placeholder-photo.svg` (на реальные фото/видео обложки).
2. Укажите ваши контакты в секции `Контакты` (в `index.html`).
